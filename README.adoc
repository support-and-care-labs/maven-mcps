= Maven MCPs

A collection of MCP (Model Context Protocol) servers designed to help with Apache Maven improvements.

== MCP Server Projects

xref:{mail-mcp-root}/README.adoc[mail-mcp]::
MCP server for accessing and analyzing Apache Maven mailing list archives (2002-present).

== External MCP Servers

We also document and integrate third-party MCP servers (Atlassian, GitHub, etc.) for accessing external services.
See xref:docs/external/index.adoc[External MCP Catalog].

== Documentation

See xref:docs/index.adoc[Documentation] for:

* Complete project documentation
* Usage and architecture guides
* External MCP references
* Architecture Decision Records

== Getting Started

Navigate to the specific MCP server directory:

[source,bash]
----
cd mail-mcp
----

Then follow the README.adoc in that directory.
