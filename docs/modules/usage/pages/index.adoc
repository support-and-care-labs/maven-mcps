= Usage Overview
ifndef::env-site[:toc:]

This section provides guides for using the Maven MCP ecosystem with LLM clients.

== What's Available

The Maven MCPs enable AI assistants to access:

* **Mail Archives** - Historical mailing list discussions (dev@maven.apache.org)
* **GitHub** - Repositories, issues, and pull requests
* **Atlassian** - Confluence documentation and Jira (historical issues)
