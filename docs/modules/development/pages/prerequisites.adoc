[[python]]
== Python 3.11+

Python 3.11 or later is required for developing mail-mcp.

Used by::
* *mail-mcp*: Primary implementation language

Installation::
* macOS: `brew install python@3.11` or https://www.python.org/downloads/[python.org]
* Linux: Use your package manager or https://www.python.org/downloads/[python.org]
* Windows: https://www.python.org/downloads/[python.org]

Verification::
[source,bash]
----
python3 --version
# Should show Python 3.11.x or later
----

Further reading::
* xref:mail-mcp:development:development.adoc[mail-mcp Development Guide]

[[poetry]]
== Poetry

Poetry is the dependency management tool for mail-mcp.

Used by::
* *mail-mcp*: Manages Python dependencies and virtual environment

Installation::
[source,bash]
----
curl -sSL https://install.python-poetry.org | python3 -
----

Or via pipx:
[source,bash]
----
pipx install poetry
----

Verification::
[source,bash]
----
poetry --version
----

Configuration (recommended)::
Configure Poetry to create virtual environments in the project directory:
[source,bash]
----
poetry config virtualenvs.in-project true
----

Further reading::
* https://python-poetry.org/docs/[Poetry Documentation]
* xref:mail-mcp:development:development.adoc[mail-mcp Development Guide]

[[nodejs]]
== Node.js (Optional)

Node.js is only required for building the documentation site locally with Antora.

Used by::
* *Documentation*: Antora site generator

Installation::
* macOS: `brew install node`
* Linux: Use your package manager or https://nodejs.org/[nodejs.org]
* Windows: https://nodejs.org/[nodejs.org]

Verification::
[source,bash]
----
node --version
npx --version
----

Usage::
[source,bash]
----
npx antora antora-local-playbook.yml
----
