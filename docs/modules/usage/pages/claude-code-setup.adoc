= Claude Code Setup
ifndef::env-site[:toc:]
ifndef::env-site[:toclevels: 2]

This guide covers installing Claude Code as your MCP client.

== Overview

https://claude.ai/code[Claude Code] is Anthropic's CLI tool for Claude that supports the Model Context Protocol (MCP).
With MCP servers configured, Claude Code gains access to mailing lists, GitHub, and Atlassian services.

NOTE: Claude Code is the primary MCP client used in this documentation, but other MCP-compatible clients (Claude Desktop, Cursor, etc.) work as well.

== Installation

=== macOS

[source,bash]
----
brew install claude
----

=== Linux / WSL

[source,bash]
----
curl -fsSL https://claude.ai/install.sh | bash
----

=== Windows

[source,bash]
----
choco install claude
----

=== Verify Installation

[source,bash]
----
claude --version
----

NOTE: For other installation methods, see the https://docs.anthropic.com/en/docs/claude-code[official Claude Code documentation].

== Basic Usage

Start an interactive session:

[source,bash]
----
claude
----

List configured MCP servers:

[source,bash]
----
claude mcp list
----

Add an MCP server:

[source,bash]
----
claude mcp add <name> [options] -- <command>
----
