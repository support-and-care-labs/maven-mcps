== Overview

This catalog lists all MCP components used in Maven support workflows, both internal (developed by us) and external (third-party).

== Internal MCPs

MCP servers we develop and maintain within this repository.

|Access to Apache Maven mailing list archives (2002-present)

|===

== Planned Components

The following components may be added in the future:

* *Web Search MCPs* - Various implementations for web research capabilities
* *Database MCPs* - Access to databases relevant to Maven infrastructure

== Adding New Components

=== Internal MCPs

. Create a new subdirectory at the repository root level
. Follow the established project structure (see xref:mail-mcp::index.adoc[mail-mcp] as example)
. Add an Antora component descriptor (`docs/antora.yml`)
. Add an overview page in this components module

=== External MCPs

. Research available MCP implementations for the target service
. Document all alternatives with comparison matrix
. Note the research date explicitly
. Include authentication methods and deployment requirements
. Add entry to this catalog
